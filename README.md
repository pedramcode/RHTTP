# RHTTP

Redis HTTP server for hybrid back-end services

## Requirements
* [CArgs](https://github.com/pedramcode/cargs)
* [HIRedis](https://github.com/redis/hiredis)
* [Libevent](https://libevent.org/)
* LibSSL `sudo apt-get install libssl-dev`
* SQLite3
